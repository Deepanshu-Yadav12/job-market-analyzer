## ✨ Features

- 🔍 **Real-Time Job Search** – Fetch live job listings by title & location
- 🃏 **HTML Job Cards** – Beautiful cards with skill tags and salary badges
- 📊 **Dashboard Tab** – Top companies & locations hiring
- 🛠 **Skills Analysis** – Most in-demand skills bar chart + pie chart
- 💰 **Salary Insights** – Average & max salary metrics by company
- 📥 **CSV Export** – Download the full dataset instantly
- 🌙 **Premium Dark Theme** – Professional data analyst UI

## 🛠 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Deepanshu-Yadav12/HireScope.git
cd HireScope

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your API keys
# Create a .env file in the root folder:
echo ADZUNA_APP_ID=your_app_id > .env
echo ADZUNA_APP_KEY=your_app_key >> .env

# 4. Run the app
streamlit run app.py
```

---

## 🔑 Get Free API Keys

1. Go to 👉 [https://developer.adzuna.com/](https://developer.adzuna.com/)
2. Sign up for a free account
3. Copy your **App ID** and **App Key**
4. Add them to your `.env` file
